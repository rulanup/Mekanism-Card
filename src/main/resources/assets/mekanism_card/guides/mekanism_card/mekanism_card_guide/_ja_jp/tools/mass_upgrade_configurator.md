# 一括アップグレード設定器

<ItemImage id="mekanism_card:mass_upgrade_configurator" />

一括アップグレード設定器は、複数の Mekanism 機械にアップグレードを一括でインストールまたはクリアできます。

<RecipeFor id="mekanism_card:mass_upgrade_configurator" />

## アップグレードの供給元

ツールを使用する前に、インベントリまたはバインドされた AE2/QIO ネットワークにアップグレードモジュールを用意しておく必要があります。

スニーク + 右クリックで元の機械のアップグレード種類と数量を保存します。インストールモードでは、インベントリ、AE2、QIO の順に素材を使用して対象を保存済み設定へ同期します。

## ネットワークストレージ

一括アップグレード設定器は、バインドされた AE2 ネットワークまたは QIO 周波数からアップグレードモジュールを取り出せます。

- AE2：Wireless Access Point のリンクスロットでバインドします。
- QIO：周波数が選択された QIO ブロックをスニーク + 右クリックしてバインドします。
- 消費優先度：インベントリ、AE2、QIO。

## モード

設定器には2つのモードがあります：

- インストールモード：対応する各アップグレードを保存数へ同期し、余分なアップグレードはプレイヤーへ返します。
- クリアモード：取り付け済みの全種類を取り外してプレイヤーに返します。

スニーク + 空中を左クリックするとモードを切り替えます。

## 操作

- 機械を右クリック：その機械に現在のモードを適用します。
- スニーク + 機械を右クリック：その機械のアップグレード設定を保存します。
- 選択キー（既定は左 Ctrl）を押しながら2つの角を左クリックすると、その直方体範囲をすぐに処理します。
- FTB Ultimine キーを押しながら機械を右クリックすると、現在の選択範囲を一括処理します。

## クイック選択

最初の角を選ぶと、照準先までの範囲が赤い枠でプレビューされます。選択キーを押したまま2つ目の角を左クリックすると実行します。

## 中クリックショートカット

- 機械を中クリック：対応するすべてのアップグレードモジュールを自動で取り付けて満杯にします。
- FTB Ultimine キーを押しながら中クリックすると、現在の選択範囲へ一括インストールします。
