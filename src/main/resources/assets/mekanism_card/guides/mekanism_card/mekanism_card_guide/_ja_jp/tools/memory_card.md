# メモリカード

<ItemImage id="mekanism_card:memory_card" />

メモリカードは、Mekanism 機械の設定をコピーして別の機械へ貼り付けるために使用します。1台の機械から設定とアップグレードを保存し、他の機械へ適用できます。

<RecipeFor id="mekanism_card:memory_card" />

## コピー

設定をコピーするには：

- スニーク + 機械を右クリック：設定とアップグレードをカードにコピーします。

ツールチップには保存元の機械タイプが表示されます。

## 貼り付け

設定を貼り付けるには：

- 機械を右クリック：保存された設定を対象の機械に適用します。必要に応じてアップグレードカードを消費します。

対象の機械タイプが一致しており、インベントリまたはバインドされた AE2/QIO ネットワークに十分なアップグレードカードが必要です。

## 一括貼り付け

- 選択キー（既定は左 Ctrl）を押しながら2つの角を左クリック：直方体内の同じ種類の機械へ一括で貼り付けます。
- FTB Ultimine キーを押しながら機械を右クリック：現在の選択範囲内の同じ種類の機械へ一括で貼り付けます。

通常の右クリックでは、クリックした1台だけに貼り付けます。

## ネットワークストレージ

メモリカードは貼り付け時に、不足しているアップグレードカードをバインドされた AE2 ネットワークまたは QIO 周波数から取り出せます。

- AE2：Wireless Access Point のリンクスロットでバインドします。
- QIO：周波数が選択された QIO ブロックをスニーク + 右クリックしてバインドします。
- 消費優先度：インベントリ、AE2、QIO。

## クリア

保存された設定をクリアするには：

- スニーク + 空中で右クリック：保存された設定をクリアします。

これでカードがリセットされ、新しい機械をコピーできます。
