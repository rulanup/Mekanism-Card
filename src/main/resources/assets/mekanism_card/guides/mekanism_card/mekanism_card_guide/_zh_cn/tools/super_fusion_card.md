# 超级融合卡

<ItemImage id="mekanism_card:super_fusion_card" />

超级融合卡将等级安装、升级模块安装和内存卡复制/粘贴合并到一个物品中。

<RecipeFor id="mekanism_card:super_fusion_card" />

## 操作方式

- Shift + 右键空气：切换模块选区模式。
- Ctrl + 右键空气：在等级安装、安装升级、内存复制/粘贴之间切换融合模式。
- 按住 G：打开模式转盘，指向目标融合模式后松开 G 完成选择。
- Ctrl + Shift + 右键空气：切换范围/单个模式。
- Alt + 右键空气：清除已保存的内存数据。
- 右键机器：执行当前融合模式，并阻止机器界面打开。
- 鼠标中键机器：自动为机器安装所有支持的升级模块（每种升到满级）。处于范围模式时会对所有相邻/选区机器执行批量安装。

## 模式

- 等级安装：将机器等级向终极升级。
- 安装升级：安装或移除升级模块。处于该模式时，右键空气切换安装/移除。
- 内存复制/粘贴：无数据时复制机器内存，有数据时粘贴内存。

## 合成

超级融合卡由钢锭、批量升级配置器、内存卡、基础能量立方、Mekanism 配置卡和锇锭合成。

## 网络存储

超级融合卡可以从绑定网络中抽取等级安装器和升级模块。

- AE2：放入无线访问点链接槽绑定。
- QIO：潜行 + 右键已选择频道的 QIO 方块绑定。
- 消耗优先级：背包、AE2、QIO。
