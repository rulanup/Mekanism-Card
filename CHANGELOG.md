# Changelog

### English

#### v2.4.1 (Current)

- **New**: Added Full Paste as the fourth Super Fusion Card mode. Precise Mode applies the copied tier, upgrade profile, and complete configuration to the same machine family; Fuzzy Mode applies tiers and compatible upgrades across machine types without machine-specific configuration.
- **Changed**: Mass Upgrade Configurator and Super Fusion Card Upgrade mode now save a machine's upgrade types and levels with Shift-right-click. Install synchronizes targets to that profile, obtains missing upgrades from inventory, AE2, or QIO, and returns surplus upgrades to the player.
- **New**: Added Precise Mode and Fuzzy Mode. Fuzzy Mode allows compatible cross-type operations, while Precise Mode only allows operations on the same machine type.
- **Controls**: Unified Memory Card, Mass Upgrade Configurator, and Ultimate Tier Installer controls with the corresponding Super Fusion Card modes. All four fusion modes and three standalone tools support live two-corner cuboid selection; removed the legacy area mode, M-key, Alt-right-click, and Ctrl-right-click workflows.
Card shortcuts are now standardized as follows:
- Left-click air: open the configuration screen.
- Shift + left-click air: cycle the fusion mode.
- Hold the selection key (Left Ctrl by default) and left-click two corners: execute a batch module operation.
- Shift + right-click a machine: copy the machine configuration to the card.
- Shift + right-click air: clear the saved configuration.
- Right-click a machine: execute the current fusion mode; paste the copied configuration in Memory mode.
- Middle-click a machine: install all supported upgrades; while holding the FTB Ultimine key, process its selection in bulk.
- **Controls**: Tier installation, upgrade profile synchronization, memory paste, Full Paste, and middle-click install-all operations support FTB Ultimine batch processing.
- **Jade**: Holding an upgrade configurator or Super Fusion Card now directly displays the machine's installed upgrades.
- **GUI/Guide/Tooltip**: Adjusted the Super Fusion Card configuration screen, exposed every tool as its own first-level guide tab, and changed card shortcut help to gray descriptions with green key combinations.

### 中文

#### v2.4.1（当前）

- **新增**：超级融合卡新增第四种“全量模式”。精准模式向同一机器家族应用复制的等级、升级配置和完整机器配置；模糊模式可跨机器类型应用等级与兼容升级，但不写入机器专属配置。
- **更改**：批量升级配置器与超级融合卡安装升级模式改为 Shift+右键保存机器的升级类型和数量。安装模式将目标同步到该配置，从背包、AE2 或 QIO 获取缺少的升级，并把多余升级返还玩家。
- **新增**：新增精准模式和模糊模式，模糊模式允许兼容的跨类型操作，精准模式仅允许同类型操作。
- **操作**：将内存卡、批量升级配置器和终极等级安装器的操作统一到超级融合卡对应模式。四种融合模式及三件独立工具均支持带实时边框的双角点长方体选区；移除旧区域模式、M 键、Alt+右键和 Ctrl+右键流程。
卡牌快捷键统一更改为：
- 左键空气：打开配置界面
- Shift + 左键空气：切换融合模式
- 按住选区键（默认左 Ctrl）并左键两个角点：执行批量模块操作
- Shift + 右键机器：将机器配置复制到卡中
- Shift + 右键空气：清除已保存的配置
- 右键机器：执行当前融合模式；内存模式下粘贴已复制配置
- 中键机器：安装所有支持的升级；按住 FTB 连锁键时批量处理其选区
- **操作**：等级安装、升级配置同步、内存粘贴、全量粘贴和中键装满升级均支持连锁批量操作。
- **Jade**：手持升级配置器或超级融合卡时可直接显示机器已安装升级。
- **GUI/指南/提示**：调整超级融合卡配置界面，指南中的每件工具均作为独立一级标签显示，卡牌快捷键帮助使用灰色说明和绿色按键组合。
