# Changelog

### English

#### v2.4.2 (Current)

- **Fixed**: Super Fusion Card Tier Installation now pastes the copied machine's factory tier instead of automatically upgrading the target to the highest available tier.
- **Fixed**: Right-clicking with the Ultimate Tier Installer no longer opens the machine interface. It now upgrades machines only to standard Ultimate without requiring tier installer items, and its ME network and QIO binding support has been removed.
- **Optimized**: Creative Mode tier pasting ignores material and energy requirements and never consumes resources.

---

#### v2.4.1

- **Added**: Full Paste mode for Super Fusion Card, combining copied tiers, upgrade levels, and machine configuration in one operation.
- **Added**: Precise Mode and Fuzzy Mode for controlling same-family and cross-type operations.
- **Added**: Jade upgrade information, FTB Ultimine batch processing, network sourcing through AE2/QIO, and independent guide tabs for every tool.
- **Optimized**: Unified the operation and shortcut behavior of Memory Card, Mass Upgrade Configurator, Ultimate Tier Installer, and Super Fusion Card.
- **Optimized**: Upgrade installation now synchronizes copied upgrade types and quantities, obtains missing upgrades from available sources, and returns surplus upgrades to the player.
- **Optimized**: Super Fusion Card configuration screens and tooltip shortcut presentation.
- **Removed**: Legacy area mode, M-key operation, Alt-right-click workflow, and Ctrl-right-click workflow.

---

### 中文

#### v2.4.2（当前）

- **修复**：超级融合卡等级安装模式现在会粘贴复制机器的工厂等级，不再自动将目标机器升级到可用的最高等级。
- **修复**：手持终极等级安装器右键不再打开机器界面，且仅可升级到终极等级，无需各等级工厂安装器；移除绑定 ME 网络和 QIO 逻辑。
- **优化**：创造模式下粘贴等级时忽略材料和能量要求，且不会消耗任何资源。

---

#### v2.4.1

- **新增**：超级融合卡全量模式，将复制的机器等级、升级数量和机器配置合并到一次操作中。
- **新增**：精准模式和模糊模式，用于控制同机器家族操作和跨类型操作。
- **新增**：Jade 升级信息显示、FTB 连锁批量处理、AE2/QIO 网络资源获取，以及每个工具独立的指南标签。
- **优化**：统一内存卡、批量升级配置器、终极等级安装器和超级融合卡的操作与快捷键逻辑。
- **优化**：升级安装现在会同步复制的升级类型和数量，从可用资源中获取缺少的升级，并将多余升级返还给玩家。
- **优化**：超级融合卡配置界面和提示框快捷键展示。
- **移除**：旧区域模式、M 键操作、Alt+右键流程和 Ctrl+右键流程。
